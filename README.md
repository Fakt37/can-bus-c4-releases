# can-bus-c4-releases — релизы приложения

Репозиторий для хранения APK-файлов и метаданных версий приложения [CAN Bus C4](https://github.com/Fakt37/can_bus_c4).

## Обновление

Приложение проверяет `version.json` при запуске и предлагает обновление если доступна новая версия.

## Формат version.json

```json
{
  "version": "1.0.5",
  "url": "https://github.com/Fakt37/can-bus-c4-releases/releases/download/v1.0.5/can-bus-c4-1.0.5.apk",
  "notes": "Описание изменений"
}
```
